# Instructions pour le logo Proxima Care

## Emplacement du fichier
Placez votre fichier logo dans le dossier `public/` avec le nom exact :
- **Fichier** : `logo.png`
- **Chemin complet** : `public/logo.png`

## Formats acceptés
- PNG (recommandé avec fond transparent)
- SVG (si disponible)
- JPG (si PNG non disponible)

## Taille recommandée
- **Minimum** : 512x512 pixels
- **Optimal** : 1024x1024 pixels ou plus
- **Format** : Carré ou rectangulaire (ratio 1:1 recommandé)

## Utilisation du logo sur le site
Le logo est utilisé dans les sections suivantes :

1. **Header (en-tête)** - Logo 48x48px avec texte
2. **Hero (première de couverture)** - Logo 200-280px (grand format)
3. **Section Présentation** - Logo 120x120px
4. **Section Services** - Logo 100x100px
5. **Section Approche** - Logo 100x100px
6. **Section Valeurs** - Logo 100x100px
7. **Section Vision** - Logo 120x120px
8. **Footer (pied de page)** - Logo 60x60px avec texte

## Notes importantes
- Le logo doit avoir un fond transparent ou blanc pour une meilleure intégration
- Les dimensions seront ajustées automatiquement par Next.js Image
- Le logo est optimisé pour le web avec `priority` sur la première image
